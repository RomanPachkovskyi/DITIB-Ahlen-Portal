# CLAUDE.md — Compatibility Pointer
## DITIB-Ahlen-Portal

У цьому репозиторії `AGENTS.md` є канонічним instruction-файлом для всіх AI-агентів, включно з Claude Code.

Claude Code має прочитати ці файли перед будь-якими змінами:

1. `AGENTS.md` — спільні правила агентів, команди і середовище
2. `PROJECT.md` — архітектура, поточний стан, deploy-процес, плани і backlog
3. `CHANGELOG.md` — хронологія змін

Не дублювати правила проекту в цьому файлі. Не створювати окремий Claude-specific workflow.

Якщо треба змінити інструкції для агентів, оновити `AGENTS.md`.

Якщо треба змінити архітектуру, стан продукту, deploy-процес, плани або backlog, оновити `PROJECT.md`.

Після будь-якої зміни в репозиторії додати запис сесії в `CHANGELOG.md`.
